# Gerador de Senhas

Este é um projeto simples em Python que gera senhas aleatórias.

## Como usar

1. Clone o repositório:
   git clone https://github.com/rcthefato
   cd nome-do-repo
Execute o script:


python gerador_senha.py
Digite o tamanho desejado da senha quando solicitado.

Requisitos
Python 3.x
